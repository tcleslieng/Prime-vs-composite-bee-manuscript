# Prime vs composite perceptual grouping manuscript

Details on data processing, statistical analysis and figures of the manuscript under peer review: "Exploring discrimination of symmetrical/asymmetrical quantities in honeybees using perceptual grouping"

https://tcleslieng.github.io/Prime-vs-composite-bee-manuscript/

This readme file describes the data files accompanying the above publication. For any further queries please contact scarlett.howard@monash.edu The following files are included:

1) "60priming.csv"

This file contains individual choices made by bees in experiment 2. The columns are as follows:

* BEEID: individual bee recruited from the hive
* CNUM: choice number
* TEST: test condition
* CHOICE: binomial choice made by bees
* TESTORDER: counterbalancing order of tests

2) "60training_testphase_data.csv"

This file contains individual choices made by bees in experiment 6 (test phase). The columns are as follows:

* BEEID: individual bee recruited from the hive
* CNUM: choice number
* TEST: test condition
* CHOICE: binomial choice made by bees
* CONDITION: training group allocation
* GROUP: counterbalancing order of tests

3) "60training_trainphase_data.csv"

This file contains individual choices made by bees in experiment 6 (training phase). The columns are as follows:

* BEEID: individual bee recruited from the hive
* CNUM: choice number
* TEST: test condition
* CHOICE: binomial choice made by bees
* CONDITION: training group allocation
* ORDER: counterbalancing order of tests

4) "control_data.csv"

This file contains individual choices made by bees in experiment 4. The columns are as follows:

* BEEID: individual bee recruited from the hive
* TEST: test condition
* CHOICE: binomial choice made by bees
* CONDITION: group condition allocation
* TRIAL: trial number

5) "prime_data.csv" 

This file contains individual choices made by bees in experiment 1. The columns are as follows:

* BEEID: individual bee recruited from the hive
* TEST: test condition
* CHOICE: binomial choice made by bees
* CONDITION: group condition allocation

6) "primegrouped_data.csv" 

This file contains individual choices made by bees in experiment 3. The columns are as follows:

* BEEID: individual bee recruited from the hive
* TEST: test condition
* CHOICE: binomial choice made by bees
* CONDITION: group condition allocation
* CNUM: choice number

7) "training_data.csv" 

This file contains individual choices made by bees in experiment 5. The columns are as follows:

* BEEID: individual bee recruited from the hive
* CHOICE: binomial choice made by bees
* NUMBER: stimulus presented
* SHAPE: shape of stimulus presented
* CONDITION: test condition
* STIMULI: training phase group allocation
